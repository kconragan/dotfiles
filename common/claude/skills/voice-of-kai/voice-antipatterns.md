# Voice Antipatterns — Banned Phrases and Dead Constructions

This is a lookup table of specific words, phrases, and constructions that have been
flagged during writing sessions. It grows over time. Load this during editing passes,
not necessarily during initial drafting.

See `voice-of-kai.md` for the deeper voice principles this list serves.

---

## The Fatal Pattern (HARD FAIL)

Any sentence that negates one framing then asserts a corrected one. If even one
appears, the output fails. Delete the negation, just state the positive claim.

- "Not X. Y."
- "This isn't X. This is Y."
- "Forget X. This is Y."
- "Less X, more Y."
- "It's not just X — it's Y."
- "That's not a constraint. It's a responsibility."

## Dead AI Language

These words and phrases signal AI-generated text. Never use them.

| Flagged | Why |
|---------|-----|
| "In today's [anything]..." | AI opening cliché |
| "It's important to note that..." / "It's worth noting..." | Throat-clearing |
| "Delve" / "Dive into" / "Unpack" | AI vocabulary tells |
| "Harness" / "Leverage" / "Utilize" | Use "use" |
| "Landscape" / "Realm" / "Robust" | Vague inflaters |
| "Game-changer" / "Cutting-edge" | Hype language |
| "Straightforward" | AI filler |
| "I'd be happy to help" | Chatbot language |
| "In order to" | Just say "to" |
| "Supercharge" / "Unlock" / "Future-proof" | AI marketing cringe |
| "10x your productivity" | Growth-hack speak |
| "The AI revolution" / "In the age of AI" | Breathless framing |

## Dead Transitions

Never use these. If you need a transition, earn it or cut it.

| Flagged | Replace with |
|---------|-------------|
| "Furthermore" / "Additionally" / "Moreover" | Cut; restructure |
| "Moving forward" / "At the end of the day" | Cut entirely |
| "To put this in perspective..." | Just put it in perspective |
| "What makes this particularly interesting is..." | Just say the interesting thing |
| "The implications here are..." | State the implications directly |
| "In other words..." | Rewrite the preceding sentence to be clearer |
| "It goes without saying..." | Then don't say it |

## Engagement Bait

Never use these. They are the written equivalent of clickbait.

| Flagged | Why |
|---------|-----|
| "Let that sink in" / "Read that again" / "Full stop" | Tells the reader how to feel |
| "This changes everything" | Almost never true |
| "Are you paying attention?" / "You're not ready for this" | Condescending |

## Dead Constructions

| Flagged | Replace with | Why |
|---------|-------------|-----|
| "from the ground up" | Name what was actually built | Cliché |
| "pioneered" | "created," "developed," "built," "invented" | Aggrandizing over precise |
| "at the intersection of X, Y, and Z" | Demonstrate through specific work | Generic positioning |
| "passionate about" | Show the passion through specifics | Empty credentialing |

## Generic Insider Claims

| Flagged | Why |
|---------|-----|
| "Here's the part nobody's talking about" | Implies privileged access to hidden truths |
| "What nobody tells you" | Same |
| Anything with "nobody" or "most people don't realize" | Use specific cohort data or named people instead |

## AI-Coded Structural Patterns

| Flagged | Why |
|---------|-----|
| Multiple tricolons across peer sections | Creates formulaic rhythm; see Mechanics #5 in voice-of-kai.md |

## Hedging Language (Context-Dependent)

These are not always wrong — but when they appear in professional or opinion writing,
check whether they're genuine uncertainty or anxiety-driven hedging.

| Flagged | When it's a problem |
|---------|-------------------|
| "I helped drive..." | When "I drove..." is accurate |
| "we were able to..." | When "we did..." is accurate |
| "I think" / "I feel like" | When the position has been earned and doesn't need softening |
| "it was clear that" | When you can say who it was clear to, directly |
